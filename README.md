### 볼타 싱크 에이전트
이 앱은 볼타 서비스의 **가져오기(Fetch)** 기능을 자동으로 실행하는 Windows 시스템 트레이 애플리케이션입니다.

브라우저를 직접 열어 가져오기 버튼을 누를 필요 없이, 저장된 로그인 세션을 이용하여 지정된 시간마다 자동으로 데이터를 동기화합니다.

---

#### 주요 기능

- Windows 시스템 트레이에서 백그라운드 실행
- 전용 볼타 브라우저 세션 관리
- Chrome에 저장된 로그인 정보를 이용한 자동 로그인
- 로그인 후 자동 Sync 옵션
- 설정한 주기에 따른 자동 Sync
- 수동 Sync 실행
- 마지막 Sync 및 다음 Sync 시간 표시
- Windows 시작 시 자동 실행 옵션

#### 시스템 요구사항

- Windows 10 / Windows 11
- Google Chrome (Stable)
- 볼타 유료계정

#### 설치

다운받은 폴더 전체를 원하는 위치에 복사합니다.  
`config`와 `logs` 폴더는 처음 실행 시 자동으로 생성됩니다.

#### 최초 실행

1. 첫 실행시 Chrome 팝업으로 볼타 로그인 창이 열립니다.
2. 로그인 후 Chrome에 `로그인 정보를 저장하시겠습니까?`팝업을 확인/클릭하면 다음 실행부터 자동 로그인 됩니다.
3. 로그인 후 최초 Sync 작업이 실행되고 (30초 이내), 이후 설정한 간격으로 자동 Sync 됩니다.
4. 이후 PC를 부팅하면 Chrome 로그인 팝업이 보인 후 자동로그인 설정이 되어있으면 바로 닫히게 됩니다.
5. 작업표시줄 메뉴 설정에서 "기타 시스템 트레이 아이콘"을 켜두시면 상태 확인이 쉽습니다.

#### 참고사항

- Google Chrome(Stable)이 설치되어 있어야 합니다.
- 일반 Chrome 브라우저와 별도의 Runtime Browser를 사용합니다.
- 볼타용 별도 프로필 사용으로 기존 Chrome 프로필에 영향을 주지 않습니다.
- 로그인정보/개인정보는 앱에 저장되지 않으며, Chrome 브라우저 내에서 로그인 정보를 관리합니다.
